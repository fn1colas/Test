# AMI Real Estate

This project contains a minimal responsive web interface for **AMI**, a modern real estate application. It includes:

- **index.html** – User-facing page with featured properties.
- **admin.html** – Simple admin panel for property management.
- Shared CSS and JavaScript in `css/style.css` and `js/script.js`.

Both pages are responsive and should display well on mobile, tablet and desktop devices.
